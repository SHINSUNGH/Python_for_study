>   > ## 변수 연산
>   >
>   >  변수에 ( + ) 나 ( - ) 를 넣어서 연산 가능하다.
>   >
>   >  예를 들면
>   >
>   >```python
>   >a = 1
>   >a = a + 1
>   >print (a)
>   >```
>   >
>   >```python
>   >a + = 1
>   >print (a)
>   >```
>   > **(위 두개의 코드는 같은 뜻)**
>   >
>   > ANS = 2
>   >  2인 값이 나오는 이유는
>   >  이미 a + 1 이라는 2인 값이 a 에 할당되기 때문
